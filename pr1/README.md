Опис

Ця програма реалізована на мові C. Спочатку користувач вводить розмір масиву. Після цього програма створює динамічний масив відповідного розміру за допомогою malloc. Далі користувач вводить число, яким заповнюється весь масив.

Програма також дозволяє змінювати розмір масиву під час виконання за допомогою функції realloc. Якщо розмір масиву збільшується, нові елементи заповнюються тим самим числом. Якщо розмір зменшується, зайві елементи видаляються.

У програмі реалізована перевірка використання пам'яті, яка показує поточний стан виділення пам'яті при операціях malloc, realloc та free.

Приклад роботи
Enter array size (1 - 100): 5
Enter number to fill array: 7

[Memory operation: malloc]
Array size: 5 elements
Allocated memory: 20 bytes

Array: 7 7 7 7 7

Enter new array size (1 - 100): 8

[Memory operation: realloc]
Array size: 8 elements
Allocated memory: 32 bytes

Array: 7 7 7 7 7 7 7 7
Обмеження

Мінімальний розмір масиву — 1 елемент

Максимальний розмір масиву — 100 елементів

Масив містить лише цілі числа

Розмір масиву перевіряється перед виділенням пам'яті

Збірка та запуск
gcc main.c -o main
./main
