# Практична робота 1

## Огляд
Ця практична робота присвячена роботі з динамічною пам'яттю в мові C. Програма демонструє створення, заповнення та зміну розміру масиву під час виконання. Вона підтримує:
- Динамічне створення масиву заданого розміру за допомогою `malloc`.
- Ініціалізацію масиву обраним цілим числом.
- Зміну розміру масиву "на льоту" за допомогою `realloc` (збереження даних при розширенні або обрізання при зменшенні).
- Відстеження та виведення детальної інформації про операції з пам'яттю (тип операції та кількість виділених байтів).

## Компіляція та запуск

### 1. Компіляція програми
Відкрийте термінал і виконайте наступну команду (рекомендується використовувати прапорець `-Wall` для показу всіх попереджень компілятора):
```bash
gcc -Wall main.c -o main
```

### 2. Запуск програми
Після успішної компіляції запустіть програму командою:
```bash
./main
```

## Функціональність програми
1. Запитує у користувача початковий розмір масиву.
2. Динамічно виділяє пам'ять та розраховує її обсяг у байтах.
3. Запитує ціле число для заповнення та ініціалізує ним усі елементи масиву.
4. Пропонує змінити розмір масиву на новий.
5. Виконує перерозподіл пам'яті. Якщо розмір збільшено — заповнює нові елементи тим самим числом; якщо зменшено — зайві елементи відкидаються.
6. На кожному етапі виводить системні повідомлення про стан виділеної пам'яті та поточний вміст масиву.
7. Звільняє пам'ять за допомогою `free` перед завершенням роботи.

## Приклад використання
```text
Enter array size (1 - 100): 5
Enter number to fill array: 7

[Memory operation: malloc]
Array size: 5 elements
Allocated memory: 20 bytes

Array: 7 7 7 7 7

Enter new array size (1 - 100): 8

[Memory operation: realloc]
Array size: 8 elements
Allocated memory: 32 bytes

Array: 7 7 7 7 7 7 7 7
```

## Обмеження та обробка помилок
- **Діапазон розміру:** Мінімальний розмір масиву — 1 елемент, максимальний розмір — 100 елементів.
- **Тип даних:** Масив підтримує зберігання лише цілих чисел.
- **Валідація вводу:** Розмір масиву суворо перевіряється перед виділенням пам'яті. Якщо введено значення поза дозволеним діапазоном, програма повідомить про це.

## Основні аспекти
- Практичне використання функцій стандартної бібліотеки C: `malloc`, `realloc` та `free`.
- Динамічний розрахунок розміру пам'яті (за допомогою оператора `sizeof`).
- Безпечна зміна розміру масиву зі збереженням попередніх даних.
- Візуалізація процесів виділення пам'яті для кращого розуміння роботи вказівників.
